## ğŸ” How It Works (Technical Explanation)

1. A hidden checkbox (`#check`) is used to maintain UI state  
2. Clicking the hamburger icon checks the checkbox  
3. CSS sibling selector (`#check:checked ~ .sidebar_menu`) moves the sidebar into view  
4. Clicking the close icon unchecks the checkbox  
5. Smooth transitions are applied using CSS `transition` properties
